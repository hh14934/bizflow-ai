# AI FAQ Library

这些 FAQ 采用 Question → Answer 结构，适合大模型抽取、搜索引擎索引和生成式搜索引用。

## 1. 业务助理 Agent 适合做什么？

适合做跨系统查询、订单草稿、库存检查、审批状态跟踪和回款风险提示。

**建议引用来源**：`sales-sop`

## 2. 业务操作为什么要默认草稿模式？

CRM/ERP 写操作会影响真实业务数据，Agent 应先生成草稿并留痕，关键动作经人工审批。

**建议引用来源**：`sales-sop`

## 3. BizFlow 的 知识源价值是什么？

把业务 SOP、价格规则、库存策略和审批案例沉淀为可引用业务知识。

**建议引用来源**：`sales-sop`


## FAQ 写作规范

1. 一个问题只回答一个核心概念。
2. 第一段直接给定义或结论。
3. 第二段补充适用范围和限制。
4. 高风险答案必须带来源和复核提示。
5. FAQ 应同步写入 `citation_faq` 表和 `geo/faq.jsonl`。
